# TRADE CHASER — 분석 카테고리 & 성향 정의

> 사용자가 "이 문서를 **어떤 카테고리 관점**에서, **어떤 성향(톤)**으로 분석할지" 고르는 렌즈의 후보 목록.
> 아래 값들은 DB `analysis_categories` / `analysis_presets` 시드로 그대로 들어간다.
> 관련: [기획서.md](기획서.md) · [DB스키마.md](DB스키마.md) · [프롬프트템플릿.md](프롬프트템플릿.md)

## 1. 카테고리 (`analysis_categories`)

`type` 4종으로 구분: **지수(index) / 지표(indicator) / 섹터(sector) / 테마(theme)**

### 1-1. 지수 (index) — 시장 전체 방향
| code | 한글명 | 설명 |
| --- | --- | --- |
| `IXIC` | 나스닥 종합 | 기술주 중심 |
| `SPX` | S&P 500 | 미국 대형주 대표 |
| `DJI` | 다우존스 산업평균 | 전통 대형주 30 |
| `RUT` | 러셀 2000 | 중소형주 |
| `SOX` | 필라델피아 반도체지수 | 반도체 업황 바로미터 |
| `VIX` | 변동성 지수(공포지수) | 시장 불안 심리 |
| `DXY` | 달러 인덱스 | 달러 강·약 (환율 영향) |

### 1-2. 지표 (indicator) — 거시 이벤트
| code | 한글명 | 설명 |
| --- | --- | --- |
| `FFR` | 미 연준 기준금리 | 통화정책의 핵심 |
| `US10Y` | 미 국채 10년물 금리 | 성장주 밸류에이션에 직결 |
| `CPI` | 소비자물가지수 | 인플레이션 |
| `PCE` | 개인소비지출 물가 | 연준이 보는 물가 |
| `NFP` | 비농업 고용/실업률 | 고용 = 금리 방향 |
| `PMI` | ISM 제조·서비스 PMI | 경기 확장/수축 |
| `GDP` | 실질 GDP 성장률 | 경기 체력 |
| `FOMC` | FOMC 회의·발언 | 금리 결정 이벤트 |

## 2. 분석 성향 / 톤 (`analysis_presets`)

"잡지 성향"에 해당. 같은 문서라도 아래 렌즈에 따라 해석 톤·강조점이 달라진다. **사실은 동일하고 서술 관점만 변함.**

| code | 이름 | 성향 설명 (프롬프트 persona로 주입) |
| --- | --- | --- |
| `MACRO` | 냉정한 거시 분석 | 감정 배제, 금리·거시·수급 중심. *The Economist* 톤. |
| `VALUE` | 장기 가치투자 | 펀더멘털·밸류에이션·현금흐름 중심, 단기 노이즈 무시. |
| `MOMENTUM` | 단기 모멘텀 | 수급·모멘텀·촉매 이벤트 중심, 단기 변동성 주목. |
| `FACT` | 팩트 브리핑 | 해석 최소화, 사실·수치·원문 인용만. |
| `BEGINNER` | 입문자용 쉬운 설명 | 용어 풀이·비유 사용, 배경부터 친절하게. |

> ⚠️ 어떤 성향이든 **매매 지시 금지 / 근거 인용 / 투자권유 아님** 가드레일은 고정 (성향으로 못 덮음).

---

## 3. 심층도 / 출력 옵션

| 옵션 | 값 |
| --- | --- |
| depth (심층도) | `brief`(3줄 요약) / `standard` / `deep`(근거·배경까지) |
| language | `ko` / `en` |
| length | 짧게 / 보통 / 길게 |

## 4. 사용자 선택 예시

> "이 엔비디아 실적 리포트를 **[SEMI · SOX · US10Y] 관점**에서 **[가치투자(VALUE)] 성향**, **심층(deep)**, **한국어**로 분석해줘"
